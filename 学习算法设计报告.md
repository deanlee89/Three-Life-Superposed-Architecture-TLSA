## 1. 数学理论基础

### 1.1 三生状态空间

三生架构的核心计算单元是三元概率单纯形 Δ²：

$$\Delta^2 = \{(α, γ, β) \in \mathbb{R}^3 : α + γ + β = 1, α, γ, β \geq 0\}$$

其中：
- **α (阴/收敛态)**: 代表向内、凝聚的能量趋势
- **γ (和/平衡态)**: 代表和谐、统一的力量
- **β (阳/发散态)**: 代表向外、扩展的能量趋势

**参数化方式**: 使用 softmax 函数确保约束满足：

$$(α, γ, β) = \text{softmax}(\theta_1, \theta_2, \theta_3) = \frac{1}{Z}(e^{\theta_1}, e^{\theta_2}, e^{\theta_3})$$

其中 $Z = e^{\theta_1} + e^{\theta_2} + e^{\theta_3}$

### 1.2 耦合算子

耦合算子定义了两个元胞状态 $(α_u, γ_u, β_u)$ 和 $(α_v, γ_v, β_v)$ 之间的交互：

#### 原始形式（用于兼容性证明）

$$\tilde{\gamma} = \alpha_v^2 \cdot \beta_u^2 + \beta_v^2 \cdot \alpha_u^2 + \lambda_h \cdot \gamma_v \cdot \gamma_u$$
$$\tilde{\alpha} = \alpha_v^2 + \alpha_v \cdot \gamma_u$$
$$\tilde{\beta} = \beta_v^2 + \beta_v \cdot \gamma_u$$

#### 增强版（用于仿真，含平衡引力）

$$\text{cross} = \alpha_v \cdot \beta_u + \beta_v \cdot \alpha_u$$

$$\gamma_{\text{raw}} = \gamma_v \cdot (\alpha_v \cdot (\alpha_v + \gamma_v) + \text{cross} \cdot 0.3) + \lambda_h \cdot \gamma_v \cdot \gamma_u$$

$$\alpha_{\text{out}} = \alpha_v^2 + \alpha_v \cdot \gamma_u + \lambda_{\text{bal}} \cdot (\alpha_u - \beta_u)$$

$$\beta_{\text{out}} = \beta_v^2 + \beta_v \cdot \gamma_u - \lambda_{\text{bal}} \cdot (\alpha_u - \beta_u)$$

其中：
- $\epsilon$: 耦合强度，控制邻居影响的权重
- $\lambda_h$: 和合系数，控制和分量的自耦合
- $\lambda_{\text{bal}}$: 平衡引力系数，驱动阴阳平衡

## 2. 梯度推导

### 2.1 损失函数

总损失函数定义为：

$$L = L_{\text{task}}(\Psi_T) + \lambda_{\text{emerge}} \cdot L_{\text{emerge}}(\Psi_T) + \lambda_{\text{bal}} \cdot L_{\text{bal}}(\Psi_T)$$

其中：
- $L_{\text{task}}$: 任务损失（交叉熵、MSE等）
- $L_{\text{emerge}}$: 和合涌现损失
- $L_{\text{bal}}$: 阴阳平衡正则

### 2.2 可学习参数梯度

设 $\Psi_t$ 为第 $t$ 步的状态，通过链式法则：

$$\frac{\partial L}{\partial \epsilon} = \frac{\partial L}{\partial \Psi_T} \cdot \frac{\partial \Psi_T}{\partial \epsilon}$$

对于参数 $p \in \{\epsilon, \lambda_h, \lambda_{\text{bal}}\}$:

$$\frac{\partial L}{\partial p} = \sum_{t=1}^{T} \frac{\partial L}{\partial \Psi_t} \cdot \frac{\partial \Psi_t}{\partial p}$$

### 2.3 非线性项的Jacobian

#### 平方项梯度

$$\frac{\partial (\alpha^2)}{\partial \alpha} = 2\alpha$$

$$\frac{\partial (\beta^2)}{\partial \beta} = 2\beta$$

#### 和分量梯度

$$\frac{\partial \gamma_{\text{raw}}}{\partial \alpha_v} = \gamma_v \cdot (2\alpha_v + \gamma_v) + \gamma_v \cdot 0.3 \cdot \beta_u$$

$$\frac{\partial \gamma_{\text{raw}}}{\partial \beta_u} = \gamma_v \cdot 0.3 \cdot \alpha_v$$

$$\frac{\partial \gamma_{\text{raw}}}{\partial \gamma_v} = \alpha_v(\alpha_v + \gamma_v) + \text{cross} \cdot 0.3 + \lambda_h \cdot \gamma_u$$

$$\frac{\partial \gamma_{\text{raw}}}{\partial \lambda_h} = \gamma_v \cdot \gamma_u$$

#### 平衡引力梯度

$$\frac{\partial \alpha_{\text{out}}}{\partial \lambda_{\text{bal}}} = \alpha_u - \beta_u$$

$$\frac{\partial \beta_{\text{out}}}{\partial \lambda_{\text{bal}}} = -(\alpha_u - \beta_u)$$

### 2.4 softmax投影的Jacobian

softmax函数 $\sigma(\theta)_i = \frac{e^{\theta_i}}{\sum_j e^{\theta_j}}$ 的Jacobian：

$$\frac{\partial \sigma_i}{\partial \theta_j} = \sigma_i (\delta_{ij} - \sigma_j)$$

## 3. 黎曼优化器设计

### 3.1 问题分析

标准 Adam 优化器在欧氏空间 $\mathbb{R}^n$ 中更新，无法保证参数始终在单纯形 $\Delta^n$ 上。通过 softmax 投影虽然可行，但在高维情况下：
1. 需要额外的投影步骤
2. 投影可能破坏优化方向
3. 效率较低

### 3.2 Fisher信息度量

概率单纯形 $\Delta^n$ 是黎曼流形，其 Fisher 信息度量定义为：

$$g_{ij}(\theta) = \mathbb{E}_{x \sim \text{Cat}(\theta)} \left[ \frac{\partial \log p(x|\theta)}{\partial \theta_i} \cdot \frac{\partial \log p(x|\theta)}{\partial \theta_j} \right] = \frac{\delta_{ij}}{\theta_i} - 1$$

### 3.3 自然梯度

自然梯度定义为：

$$\tilde{\nabla}L = G^{-1}(\theta) \cdot \nabla_{\theta}L$$

其中 $G(\theta)$ 是 Fisher 信息矩阵。

### 3.4 SimplexAdam 算法

```
Algorithm: SimplexAdam
Input: 参数 θ ∈ Δ^n, 梯度 g, 学习率 α, 动量参数 β1, β2

1. 将梯度变换到对数空间:
   g_log = g ⊙ θ

2. Adam 动量更新:
   m ← β1 · m + (1 - β1) · g_log
   v ← β2 · v + (1 - β2) · g_log²

3. 偏差修正:
   m̂ = m / (1 - β1^t)
   v̂ = v / (1 - β2^t)

4. 计算更新步:
   Δ = m̂ / (√v̂ + ε)

5. 更新并投影:
   θ ← softmax(θ - α · Δ)

Output: θ ∈ Δ^n
```

## 5. 代码设计

### 5.1 SanshengLayer 核心实现

```python
class SanshengLayer(nn.Module):
    def __init__(self, grid_size, epsilon=0.5, lambda_h=0.3, lambda_bal=0.1, num_steps=3):
        super().__init__()
        # 可学习参数（对数尺度）
        self.log_epsilon = nn.Parameter(torch.tensor(0.0))
        self.log_lambda_h = nn.Parameter(torch.tensor(-1.0))
        self.log_lambda_bal = nn.Parameter(torch.tensor(-2.0))
        self.num_steps = num_steps
        
    @property
    def epsilon(self):
        return F.softplus(self.log_epsilon) + 1e-8
        
    def _coupling_operator(self, x_u, x_v):
        """核心耦合算子"""
        alpha_u, gamma_u, beta_u = x_u[..., 0], x_u[..., 1], x_u[..., 2]
        alpha_v, gamma_v, beta_v = x_v[..., 0], x_v[..., 1], x_v[..., 2]
        
        # 交叉项
        cross = alpha_v * beta_u + beta_v * alpha_u
        
        # 和分量
        gamma_raw = gamma_v * (alpha_v * (alpha_v + gamma_v) + cross * 0.3)
        gamma_raw = gamma_raw + self.lambda_h * gamma_v * gamma_u
        
        # 阴阳分量
        alpha_out = alpha_v**2 + alpha_v * gamma_u
        beta_out = beta_v**2 + beta_v * gamma_u
        
        # 平衡引力
        balance_force = self.lambda_bal * (alpha_u - beta_u)
        alpha_out = alpha_out + balance_force
        beta_out = beta_out - balance_force
        
        # 投影到单纯形
        raw = torch.stack([alpha_out, gamma_raw, beta_out], dim=-1)
        return F.softmax(raw, dim=-1)
    
    def forward(self, x):
        x = F.softmax(x, dim=-1)  # 确保在单纯形上
        for _ in range(self.num_steps):
            neighbors = self._get_neighbors(x)
            # 耦合演化...
            x = self._coupling_step(x, neighbors)
        return x
```

### 5.2 SimplexAdam 优化器

```python
class SimplexAdam(optim.Optimizer):
    def __init__(self, params, lr=1e-3, betas=(0.9, 0.999), eps=1e-8):
        defaults = dict(lr=lr, betas=betas, eps=eps)
        super().__init__(params, defaults)
    
    def step(self, closure=None):
        for group in self.param_groups:
            for p in group['params']:
                if p.grad is None:
                    continue
                    
                grad = p.grad.data
                state = self.state[p]
                
                # 初始化动量
                if len(state) == 0:
                    state['exp_avg'] = torch.zeros_like(p.data)
                    state['exp_avg_sq'] = torch.zeros_like(p.data)
                
                # 1. 对数空间梯度
                log_grad = grad * p.data.clamp(min=1e-8)
                
                # 2. Adam动量
                exp_avg, exp_avg_sq = state['exp_avg'], state['exp_avg_sq']
                beta1, beta2 = group['betas']
                exp_avg.mul_(beta1).add_(log_grad, alpha=1-beta1)
                exp_avg_sq.mul_(beta2).addcmul_(log_grad, log_grad, value=1-beta2)
                
                # 3. 更新
                denom = (exp_avg_sq.sqrt() / (1-beta2**0.5)).add_(group['eps'])
                step = exp_avg / denom
                p.data.add_(step, alpha=-group['lr'])
                
                # 4. 投影回单纯形
                with torch.no_grad():
                    p.data[:] = F.softmax(p.data, dim=-1)
```

### 5.3 梯度验证代码

```python
def verify_gradients(model, x, target, rtol=1e-4):
    """有限差分法验证梯度"""
    
    # 自动微分梯度
    output = model(x)
    loss = F.mse_loss(output, target)
    loss.backward()
    autograd_grad = model.log_epsilon.grad.clone()
    
    # 数值梯度
    h = 1e-6
    model.log_epsilon.data += h
    output_plus = model(x)
    loss_plus = F.mse_loss(output_plus, target)
    
    model.log_epsilon.data -= 2*h
    output_minus = model(x)
    loss_minus = F.mse_loss(output_minus, target)
    
    model.log_epsilon.data += h
    
    numerical_grad = (loss_plus - loss_minus) / (2 * h)
    
    # 相对误差
    rel_error = abs(numerical_grad - autograd_grad.mean()) / abs(autograd_grad.mean() + 1e-8)
    
    return rel_error < rtol
```
