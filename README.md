## 一句话定位

三生架构是一项**全新的计算范式探索**——基于概率单纯形 Δ² 上的非线性耦合张量网络架构，试图在经典AI（二）与量子计算（万物）之间构建一座原理性的"桥梁"。它既不是Transformer的替代品，也不是量子计算机的竞争对手，而是一种在计算架构层面具有独立价值的中间态探索。

> ⚠️ **请注意**：本项目目前处于**早期理论研究阶段**，所有实验均在合成数据上完成，尚未在公认 benchmark 上验证，也未与标准方法（MLP、CNN 等）进行过系统对比。

1. **叠加 (Superposition)**：每个计算单元维护三元概率向量 (α, γ, β) ∈ Δ²，编码阴-阳-和合的叠加态
2. **纠缠 (Entanglement)**：耦合算子 C(ψ_v, ψ_u) 在相邻元胞间建立非线性关联
3. **演化 (Evolution)**：五行生克动力学驱动多尺度调控，脏腑自愈机制保证系统鲁棒性

## 理论创新点

> ⚠️ **重要说明**：以下"创新"更准确地说是**架构范式创新**而非原子数学操作的创新。三生架构的数学原子操作——叠加=线性组合、纠缠=张量积、演化=迭代映射——本身是标准张量运算。真正的贡献在于以"概率单纯形状态空间 + 非线性耦合 + 五行多尺度调控"的全新架构范式将这些操作重新组织，正如CNN以"局部感受野+权重共享+逐层抽象"的架构范式重组标准线性代数操作一样。这种从"函数逼近视角"到"动力系统视角"的转变，才是三生架构的创新本质。

1. **三元概率单纯形 Δ²**：每个计算单元状态 (α, γ, β) ∈ Δ² ，经典概率框架，可在经典硬件运行
2. **非线性耦合算子 C**：超越标准张量网络 (MPS/PEPS/MERA) 的表达能力；证明：线性 TN ⊂ 三生张量网络
3. **二→三涌现观察**：纯二进制初始状态经一步耦合观察到和合态，已通过数值验证
4. **Banach 压缩性证明**：当 λ_bal > 2λ_coup 时，耦合算子为压缩映射，保证收敛性
5. **图灵完备性**：通过模拟 Rule 110 证明三生 CA 图灵完备
6. **MERA-AdS/CFT 全息桥接**：三层耦合结构对应 UV/中间/IR 物理
7. **桥梁定位证明**：Transformer ⊂ 三生 ⊂ 开放量子系统

## 诚实声明：当前阶段的局限性

本项目目前处于**早期研究/概念验证阶段**：

- 所有分类实验均在**人造合成图案**（6类8×8人工图案）上完成
- **未在任何公认 benchmark（MNIST、CIFAR-10、ImageNet 等）上验证**
- **未与 MLP、CNN、标准张量网络分类器等 baseline 系统对比**
- 无 GPU 实现；当前仅有 NumPy 和实验性 PyTorch 代码
- 数学内核是标准张量运算；创新在于**架构范式的组织方式**

我们不声称三生架构已超越任何现有方法，但真诚地认为它提供了一个值得探索的有趣概念框架，在噪声鲁棒性上展现了初步潜力。
