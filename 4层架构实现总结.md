## 🏗️ 4层架构设计

### 架构流程图

```
输入 x ∈ ℝ⁶⁴
    ↓
┌─────────────────────────────────────┐
│ 第1层：编码-路由                     │
│ 六书编码 + 孙子兵法路由              │
│ Input: x                            │
│ Output: e ∈ ℝ¹², route_prob ∈ Δ⁴  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 第2层：三生耦合-法则                 │
│ 虚实耦合 + 黄帝四经约束              │
│ Input: e, route_prob                │
│ Output: ψ ∈ Δ²                     │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 第3层：拓扑多尺度                    │
│ 经络+易经+阴符 (自适应+变换+压缩)   │
│ Input: ψ                            │
│ Output: Y_compressed ∈ ℝ⁸          │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 第4层：自组织-收敛                   │
│ 正则化+道德经+收敛检查               │
│ Input: Y_compressed                 │
│ Output: y ∈ Δ² (分类概率)          │
└─────────────────────────────────────┘
    ↓
输出 y ∈ ℝ²
```

### 核心创新保留

✅ **三生耦合**（虚实动态耦合）
  - 实耦合：S = ψ_v ⊗ ψ_u
  - 虚耦合：T = grad(e)
  - 自适应混合：ε·S + α_virtual·T

✅ **经络拓扑**（自适应连接）
  - 正经（局部连接）
  - 奇脉（长程连接）
  - 补泻动力学

✅ **易经变换**（卦象注意力）
  - 8个卦象基向量
  - 纠缠谱自适应选择

✅ **阴符收缩**（选择性压缩）
  - 五贼机制：观、掩、取、舍、化
  - 15%关键位选取

✅ **序参量自组织**（道德经+庄子）
  - 有序度检测
  - 收缩-展开振荡
  - 矛盾时退守中心

### 基本使用

```python
from sansheng_4layer import SanshengCore4Layer
import numpy as np

# 创建模型
model = SanshengCore4Layer(
    input_dim=64,    # 输入维度
    n_routes=4,      # 路由数
    system_dim=8,    # 系统尺寸
    output_dim=2     # 输出维度（分类）

# 前向传播
x = np.random.randn(64)
y, info = model.forward(x)

# 输出
print(f"预测：{y}")  # [0.5018, 0.4982]
print(f"序参量：{info['order_parameter']}")
print(f"收敛：{info['is_converged']}")
```

## 💡 设计亮点

### 1. 层级合并的合理性

```
11层 ─────→ 4层

编码+路由    = 第1层 (数据入口)
             ↓
耦合+法则    = 第2层 (核心计算)
             ↓
拓扑+变换+压缩 = 第3层 (多尺度处理)
             ↓
正则+自组织+收敛 = 第4层 (演化与控制)
```

每次合并都保留了完整的功能，仅是代码组织方式改变。

### 2. 数学完整性

所有公式保持不变：
- 三态分布 ψ ∈ Δ²
- 虚实耦合 C = ε(S + αT)
- 法则约束 L_total = L_task + λ·L_reg
- 序参量 order = ||ψ̄||

### 3. 通用性

4层架构可支持：
- 图像分类（MNIST、CIFAR）
- 自然语言处理（文本分类）
- 时间序列预测
- 降维与表示学习
